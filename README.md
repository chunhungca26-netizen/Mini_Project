# Dự án Quản lý nhân viên và Tiền lương (Topic 8)
Sinh viên thực hiện: Chu Thi Nhung

## Giới thiệu
Ứng dụng Console giúp quản lý danh sách nhân viên, tính lương tự động và lưu trữ dữ liệu bền vững.

## Các chức năng chính
- Thêm, hiển thị bảng, tìm kiếm theo tên, sắp xếp theo lương.
- Xử lý tệp tin (.txt) để không mất dữ liệu khi thoát.
- Thống kê quỹ lương và lương trung bình.

## Cách chạy dự án
1. Cài đặt Python (phiên bản 3.x).
2. Mở Terminal tại thư mục dự án.
3. Chạy lệnh: `python main.py`
